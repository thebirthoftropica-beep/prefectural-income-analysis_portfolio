## プロジェクト概要
本プロジェクトは、現代日本における「学力格差」の構造を、都道府県民所得と**全国学力・学習状況調査（全国学力テスト）**のデータを用いて実証的に分析したデータジャーナリズムの実践例です。単なる相関の提示に留まらず、特定地域（島根県など）の残差分析を通じて、数値の裏側に潜む社会的・構造的な要因を考察しています。
<br>
<br>

## 主な知見
- 所得と学力の有意な相関: 単回帰分析の結果、$p < 0.01$ の水準で、県民所得と学力スコアには統計的に有意な正の相関が認められました。
- 決定係数 R^2 ≒ 0.3$: 都道府県間の学力差の約30%は所得水準で説明可能ですが、残りの70%は教育政策や文化資本（Bourdieu）といった他の変数に依存していることを示唆しています。
- 残差分析による地域特性の可視化: 島根県の負の残差（期待値より低いスコア）や、教育県とされる地域の特異な動向について、地域報道の現場視点から考察を加えています。
<br>
<br>

## 背景と動機
私は新聞社での記者・市場リサーチ業務に従事する傍ら、社会学（P.ブルデュー等）や分析哲学をベースとした社会事象の分析を続けてきました。現場での「取材」という質的調査を、統計学という「量的調査」で相対化・補強することで、より公共性の高いデータジャーナリズムを実践することを目的としています。
